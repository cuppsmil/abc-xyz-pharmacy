# 💊 ABC-XYZ Analysis — Pharmacy Retail

## 🧠 Overview

В данном проекте проведён **ABC-XYZ анализ ассортимента аптечной сети** для оптимизации управления запасами, повышения прибыльности и снижения издержек.

Анализ позволяет:

* выделить наиболее прибыльные товары
* оценить стабильность спроса
* принять решения по управлению ассортиментом

---

## 🎯 Business Goal

Оптимизировать ассортимент и управление запасами за счёт:

* фокусировки на ключевых товарах
* снижения избыточных запасов
* повышения оборачиваемости

---

## 📊 Methodology

### 🔴 ABC-анализ

Классификация товаров по вкладу в бизнес-метрики:

* **A** — ~80% выручки / прибыли
* **B** — ~15%
* **C** — ~5%

📌 Используемые показатели:

* количество продаж
* выручка
* маржинальность

---

### 🔵 XYZ-анализ

Классификация по стабильности спроса:

* **X** — стабильный спрос
* **Y** — умеренные колебания
* **Z** — нестабильный спрос

📌 Основано на вариации продаж (коэффициент вариации)

---

## 🧮 Data & Tools

* SQL — подготовка и агрегация данных
* Excel — расчёты и визуализация
* Pivot Tables — сегментация и анализ

---

## 📈 Key Insights

* Небольшая доля товаров (A) генерирует основную прибыль
* Товары категории **AX** — ключевые для бизнеса (высокая прибыль + стабильный спрос)
* Категория **CZ** — кандидаты на вывод из ассортимента
* Товары **BY / BZ** требуют оптимизации закупок

## 🚀 Business Recommendations

### 🔴 AX — удерживать и масштабировать

* приоритет в закупках
* минимальные stock-out

### 🟡 BY / BZ — оптимизировать

* уменьшить запасы
* анализировать спрос

### ⚫ CZ — пересмотреть ассортимент

* возможное исключение
* минимизация закупок
